## Relacional x Não relacional

```
Os bancos relacionais demandam um forte configuração de: tabelas, colunas e relações entre tabelas para o seu funcionamento.
```

```
Os não relacionais não são rigorosos quanto a isso, podemos criar coluna quando um dado é inserido.
```

```
O que gera flexibilidade para o não relacional e também pode ser sinônimo de desorganização.
```

```
Apesar do nome, o não relacional pode ter relações entre collections
```
