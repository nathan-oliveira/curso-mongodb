## Operador $inc

```
O operador $inc pode acrescentar ou diminuir uma quantidade especificada a um valor
```

```
Exemplo: db.blog.updateOne({ author: "Nathan" }, { $inc: { postCount: 2 } })
```

```
No exemplo, adicionamos 2 a quantidade de postCount
```
